# CHANGELOG

## [v2.2.0](https://github.com/pantsman-jp/hon-log/releases/tag/v2.2.0) (2026-05-01)

### 追加

- 貸出傾向分析機能を追加
  - 著者別貸出頻度グラフ
  - 月別貸出冊数推移グラフ
  - 統計画面はメイン画面の「統計」ボタンから起動可能

### 改善

- 可視化機能の実装により、貸出データの傾向分析が直感的に行えるように改善
- データが存在しない場合は適切なメッセージを表示し、ユーザー体験を向上

## [v2.1.3](https://github.com/pantsman-jp/hon-log/releases/tag/v2.1.3) (2026-05-01)

### 改善

- APIアクセスのレート制御を導入し、短時間の過剰アクセスによるブロックリスクを低減
  - 共有のリクエスト間隔を設定して外部サービスへの呼び出しを安定化
  - 取得済みHTMLや書影検索結果にキャッシュを追加し、同一リソースへの再アクセスを削減
- 書影取得およびISBN解析処理の効率を向上
  - `get_html` / `get_isbn` のキャッシュ化により、同一URLの二重フェッチを防止
  - 画像ダウンロードで `requests` の共有モジュールを利用し、接続オーバーヘッドを抑制
- アプリ内部の動作効率を改善
  - SQLite を WAL モードで開き、同期モードを `NORMAL` に設定して書き込み性能を向上
  - CSVインポート処理で `executemany` を利用する一括挿入に変更し、データ登録を高速化
  - `BookWidget` の再利用可能なデフォルト画像をキャッシュ化して描画負荷を軽減

### 変更

- ネットワーク処理を `src/network.py` に集約し、全HTTPアクセスに共通のレート制御を適用
- 外部API取得処理をキャッシュ付きにし、同一リクエストの重複回避とレスポンス待ち時間の削減を実現

## [v2.1.2](https://github.com/pantsman-jp/hon-log/releases/tag/v2.1.2) (2026-04-25)

### 改善

- 書影取得処理のレート制御を最適化
  - 並列実行数を 10 → 5 → 3 に段階的に削減し、外部APIへの過剰アクセスを抑制
  - 各リクエスト前に短時間の待機（sleep）を導入し、レート制限およびBot判定の回避を強化
- 書影取得ソースの優先順位を調整
  - Google Books API を優先し、NDL へのアクセス頻度を低減することで全体の成功率を向上

### 修正

- 国立国会図書館（NDL）サムネイル取得時に 403 エラーが発生する問題を修正
  - Referer ヘッダを付与することでアクセス拒否を回避
- NDL が返却するエラーレスポンス（HTTP 200 かつ小サイズデータ）を正常画像として扱っていた問題を修正
  - レスポンスサイズに基づくバリデーションを追加し、不正データを除外

### 変更

- 書影取得ロジックの耐障害性を強化
  - 外部APIの制限やブロックを考慮した設計へ調整し、安定した取得動作を実現
  - 実運用における取得成功率のばらつきを低減

## [v2.1.1](https://github.com/pantsman-jp/hon-log/releases/tag/v2.1.1) (2026-04-23)

### 修正

- Open Library 等の外部 API が「画像なし」の際に返却するプレースホルダー画像を正常な書影として誤判定していた問題
- 取得データのファイルサイズに基づく厳格なバリデーションを追加し、無効なデータによる上書きを防止
- これにより、NDL や Google Books からの有効な書影が正しく優先・保持されるようになり、全体の取得率が改善

## [v2.1.0](https://github.com/pantsman-jp/hon-log/releases/tag/v2.1.0) (2026-04-22)

### 改善

- 書影取得ソースの拡充（マルチソース化）
  - 従来の国立国会図書館（NDL）に加えて、Google Books API および Open Library Covers API を統合
  - NDLに未登録の学術書や海外書籍などの書影取得率を大幅に向上
  - APIキーを必要としない公開エンドポイントを利用する設計により、安全性を確保
- ISBN 抽出精度の向上
  - ISBN-13 に加え、古い書籍に対応する ISBN-10（末尾が X のケースを含む）の抽出をサポート
  - 正規表現を最適化し、大学図書館 OPAC の HTML ソースからの取得漏れを低減

### 変更

- データベーススキーマの自動マイグレーション
  - アプリ起動時にデータベース構造をチェックし、不足しているカラム（`volume`, `published_at` など）を自動追加するロジックを実装
  - 既存のデータを保持したまま、新機能（巻情報や出版年月表示）が利用可能
- UI 表示項目の追加
  - 書籍詳細画面にて、「巻情報」および「出版年月」の表示に対応

### 修正

- インポート処理の安定化
  - ネットワーク遅延や API エラーが発生した場合でも、アプリケーションが中断せず次の書籍の処理へ進むよう例外処理を強化
  - 複数 API を参照する際のタイムアウト設定を最適化

## [v2.0.0](https://github.com/pantsman-jp/hon-log/releases/tag/v2.0.0) (2026-04-20)
